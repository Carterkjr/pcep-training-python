# Week 1 – Getting Started with Python

This folder contains my completed work for **Week 1** of my PCEP (Certified Entry-Level Python Programmer) training plan.

The focus of this week was to establish a solid foundation in Python by understanding the development environment, basic syntax, and how Python programs are executed.

---

## Topics Covered

- Setting up the Python development environment
- Using the Python interpreter vs running Python scripts
- Basic Python syntax and program structure
- Reading and reflecting on *The Zen of Python*
- Reinforcing concepts through recap, practice ideas, and self-assessment

---

## Files Included

- **Day1-2_Setting_Up_Your_Environment.py**  
  Initial environment setup concepts and first Python interactions.

- **Day3-4_Python_Interpreter_vs_Scripts.py**  
  Exploration of interactive Python vs script-based execution.

- **Day5-6_BasicSyntax_and_Structure.py**  
  Practice with Python syntax, statements, and basic structure.

- **Day7_The_Zen_of_Python.py**  
  Review and reflection on Python’s guiding principles.

- **Day7_Recap.py**  
  End-of-week recap consolidating key concepts.

- **Week1_Practice_Ideas.py**  
  Additional practice ideas to reinforce learning.

- **Quick_Quiz.py**  
  Short quiz-style questions for knowledge checks.

- **Week1_Self_Assessment.py**  
  Self-evaluation of understanding and readiness to move forward.

---

## Learning Outcome

By the end of this week, I gained confidence in:
- Navigating the Python environment
- Understanding how Python code is executed
- Writing and reading simple Python scripts
- Applying Python’s design philosophy to future learning

This foundation supports more advanced topics in subsequent weeks, including control flow, operators, and logical expressions.
